# Glossario - Vitrine MOOC Ifes

Termos do dominio. Fonte canonica.

| Termo | Significado |
|-------|-------------|
| MOOC | Massive Open Online Course. Curso aberto, online e gratuito, sem pre-requisitos de matricula. |
| Vitrine | Site catalogo que apresenta e divulga os cursos abertos do Ifes e direciona o aluno para a inscricao. |
| Ifes | Instituto Federal do Espirito Santo. |
| CEFOR | Centro de Referencia em Formacao e em Educacao a Distancia do Ifes. Responsavel pelos MOOCs. |
| CGTI | Coordenadoria responsavel por infraestrutura, servidores, DNS e ambientes. |
| CGTE | Coordenadoria responsavel pelo desenvolvimento (tema, templates, codigo). |
| Base de Conhecimento | Projeto anterior cujo modelo de estrutura de repositorio sera replicado na Vitrine. |
| AI/UX Studio | Ferramenta onde foram produzidas as interfaces (assets de design) da nova Vitrine. |
| Dump | Copia/snapshot do banco de dados usado para trabalho local com dados reais. |
| Master | Branch de producao. PRs nunca vao direto para ela. |
| Ambiente de teste | VM dedicada da CGTI com WordPress 7 e banco de teste, onde o projeto e validado antes da producao. |
